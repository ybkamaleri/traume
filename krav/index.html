Betingelsen

For å kunne benytte NTR resultat kreves det at man har tilgang eller installert programvarer som er listet nedenfor. Alle programvarene på listen er gratis å laste ned og bruke.

R

R er et statistikprogram som benyttes i alle beregningene og figur konstruksjon i denne resultattjenesten. Programvaren kan lastes ned her.

Integrert utviklingsmiljø (IDE) verktøy

IDE benyttes til å forekle arbeidet med feilsøking eller kjøre koder. Blant de vanlige IDE som man kan velge er:

RStudio

RStudio kan lastet ned her. Ved å installerer RStudio har man også fått tilgang til R uten at man behove å installere separat. Det er mulig å få tilgang til RStudio i Oslo universitetssykehus ved å søke hos Sykehuspartner.

Emacs

Emacs krever litt ekstra innsats for å begynne med, men er veldig fleksible når man har lært grunlagt fuksjonene. R må installeres seperate når man skal benytte Emacs.

Notepad++

Notepad++ behover ingen tilpasning og enklere å bruke. R må installeres seperate når man benytter Notepad++.

LaTeX kompilator

LaTeX kopilator brukes til å produsere virksomhet rapport i PDF format. Hvis det ikke er behov å benytte denne fuksjonen så kan denne utelates. De finnes forskjellige programvarer tilgjenglige, bl.a:

TexLive

Denne programvare passer til alle operativsystemmer dvs. Linux, Windows og Mac. Programvaren kan lastet ned her.

MiKTeX

Denne programvaren passer til bare Windows operativsystem. Denne kan lastet ned her.
